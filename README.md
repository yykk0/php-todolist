1.1 TODOリストアプリ
内容: ユーザーがタスクを追加・編集・削除できるシンプルなTODOリスト。

学べること: フォーム処理、データの保存（ファイルまたは簡単なDB）、ユーザー入力のバリデーション。

ポイント: セッション管理や簡単なデータベース操作（MySQLやSQLiteなど）を練習するのに最適。
